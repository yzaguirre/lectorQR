# lectorQR
Android QR reader
## Development
Clone
```bash
git clone git@github.com:yzaguirre/lectorQR.git
```
Cordova plugins
```bash
cordova plugin add https://github.com/wildabeast/BarcodeScanner.git
```
Add android platform
```bash
ionic platform add android
```
Download www/lib dependencies
```bash
bower install
```
## Device
```bash
ionic run android
```
